จากการทำ Lab นี้ ผมได้เรียนรู้วิธีการใช้ External CSS มาเชื่อมต่อเพื่อตกแต่งหน้า HTML ให้ดูเป็นระเบียบและสวยงามขึ้น โดยเฉพาะการจัดการ Box Model ที่ช่วยให้เรากำหนดระยะห่าง padding และ margin ของ Profile Card ได้แม่นยำ
นอกจากนี้ผมยังได้ฝึกใช้ Flexbox ในการจัดเรียงส่วน Skills ให้เป็นระเบียบและไหลตามหน้าจอได้แบบอัตโนมัติ พร้อมกับลองเล่นเรื่อง Typography และการใส่ hover effect ให้กับลิงก์ต่าง ๆ เพื่อให้หน้าเว็บดูมีลูกเล่นและใช้งานได้ง่ายขึ้น
ที่สำคัญที่สุดคือการเขียน Media Query เพื่อทำ Responsive Design ให้หน้าเว็บของเรายังดูดีและอ่านง่ายอยู่เสมอไม่ว่าจะเปิดดูผ่านคอมพิวเตอร์หรือโทรศัพท์มือถือครับ
